# ゲートチェックリストの使い方

各 Gate は「次のステップに進む前に確認すべき項目」のリストです。
チェックリストを埋めるのは**人間**の判断です。AI に評価させることもできますが、最終判断は人間が行ってください。

## 使い方

1. 該当するステップの作業が終わったら Gate ファイルを開く
2. 各項目を確認し、`[ ]` を `[x]` に変えていく
3. 全項目が通過できない場合は追加調査して再確認する
4. 全項目（または「必須」の項目）が通過したら次のステップへ進む

## チェックできない項目があった場合

- **該当しない**: 「このシステムには UI がない」など、構造上適用されない場合はスキップして理由をメモする
- **まだ調査中**: 調査が必要な場合は作業を続けてから再確認する
- **情報が取得できない**: 取得できない理由（アクセス権限なし、担当者不明 等）をメモして、既知の制約として記録する
